# flamez.tech
Contains a Web Application build with **FLASK (Python Microframework, http://flask.pocoo.org/docs/1.0/)**.
> You got to start from the bottom...

Technologies used:

- **Backend**
  - Python
  - Flask
  - MySQL (SQLAlchemy Flask Module)

- **Frontend**
  - Javascript
  - HTML -> Jinja2 (Standard Template Engine Flask)
  - CSS
